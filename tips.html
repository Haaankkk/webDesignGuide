Tips for Web Design

Please check your web after you wrote the code!

Tip one: Please check carefully that your main page's name is "index.html". If you have other pages in your website, you should create links in "index.html"

Tip two: Please check carefully that your website is responsive. When your photo loads incorrectly, make sure there is alternative text at the location of the photo.

Tip three: Please read your HTML, CSS, and Javascript(if you have) files again. Use websites to check if your syntax is correct.

Tip four: Please check your links! Make sure that a user can go to every page of the website and go back to the main page by clicking links.

Tip five: If you have a client, please check your requirments. Make sure that you have already fulfilled all the requirments.

Tip six: Finally, please check your colors and layouts. A good design can gurantee that users can use this website comfortably.
