HTML

The full name of HTML is HyperText Markup Language. The main function of this language is to build the general framework of a website and fill it with text and image content. (click "HTML" in the menu bar for more information!)

CSS

The full name of CSS is Cascading Style Sheet. The main function of this language is to layout and decorate the entire page. (click "CSS" in the menu bar for more information!)
