CSS Guide

CSS helps you to decorate and manage your web!

Basic Syntax and Adding to your HTML:

CSS uses selectors. Each selector in CSS has one or more property-value pairs. Please use external stylesheet to connect your HTML and CSS!

Elements Type:

We have two types of element in CSS: inline and block. Besides, we can combine these two (inline-block) to create an element that has the property of inline element and block element.

The Position Property:

There are five types of position property: static, relative, absolute, fixed, and sticky. You will use static and relative more often. The default type is "static" (everything will just start at the upper left-hand corner). If you want to decide the location by your own, please use "relative."

Floats:

Floats help you to insert pictures in next to texts. You can also use it to create sidebars. Remember to clear floats!

Responsive Design:

If you want to create a responsive web, you need to consider flowing methods: flexible layout, scalable media, and media queries.

Flexbox:

A way to organize your containers. Your items in that container can be organized easily by using flexbox. Try it: "display: flex;"

Grid:

Similar to flexbox, but this time you can organize your items in two dimensions. A better way to use grid is to combine grid and media queries.

Aesthetics:

You can edit the color and font of your web. Also, you can add background gradient and filters.

Transitions, Transifroms, and Animations:

You can use transition to add a transition to your property. You can use transform to change the shape and location of an element. And, you can use animation to create different more complex effects.
