HTML Elements

A single HTML document consists of many elements.
An element is always started and ended by tags. Between the opening tag and the closing tag is the content.
We can add attributes that contain extra information in the opening tag and even nest some elements together.

Element Structure

< Elementname Attribute > content < / Elementname >

Specific Element Names

For headers: h1, h2, h3, h4, h5, and h6.
For paragraph: p.
For images: img with attributes src and alt.
For links: a with attribute href.
For unordered lists: ul. For ordered lists: ol. For list item: li.
There are also specific element names to distinguish the whole flow of a website, including header, nav, main, footer.
When writing the HTML file, it is always better to use these elements than using the general div elements.

Ways to add content outside HTML

Instead of putting images and links directly into our HTML files, we use their paths to represent them.
There are two types of paths.
Relative path indicates the place within our root directary.
Absolute path leads to the location in the Internet.
Relative path is more stable, while absolute path is easier to find.

How to check up markup of HTML documents

We need to check the syntax of our codes after finishing coding.

Comments

Comments are important in HTML to clarify the meaning of the code.
In HTML, we simply use "< !--" and " -->" to add comments.
